# 🚀 AI Crowd Intelligence Assistant

## 🧠 Overview
An AI-powered crowd navigation and decision-making system designed for large-scale environments like temples, stadiums, and events.

It doesn’t just show routes — it **analyzes, predicts, and guides users in real-time**.

---

## ❗ Problem
In crowded places, users face:
- Confusion in navigation  
- Long and unpredictable waiting times  
- Heavy congestion  
- Lack of real-time guidance  

---

## 💡 Solution
A smart assistant that:
- Understands user intent  
- Analyzes crowd conditions  
- Predicts future congestion  
- Recommends optimal routes  

---

## 🔥 Key Features

### 🧭 Intelligent Decision Engine
- Chooses best route based on time + crowd + prediction  

### 🔮 Predictive Intelligence
- Anticipates future crowd conditions  

### ⚡ Real-Time Simulation
- Dynamic updates and rerouting  

### 🤖 AI Assistant Guidance
- Human-like step-by-step navigation  

### 🚨 Emergency Mode
- Instant redirection to nearest exit  

---

## ⚙️ How It Works

1. User selects intent (Darshan / Food / Exit / Emergency)  
2. System evaluates multiple routes  
3. Applies scoring:
   - Time  
   - Crowd  
   - Prediction  
4. Recommends best route  
5. Continuously updates guidance
